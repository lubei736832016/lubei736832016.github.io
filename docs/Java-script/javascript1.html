    # 7-WeakMap and WeakSet（弱映射和弱集合）

    • 这些映射集合 只注重了 对象 在存储 后的 可达性是不是可变的
    • 普通的 map 里面的key 是对象后 当对象 赋值为 null 的时候 会被垃圾对象回收 ,但是 map里面的数据不会
    • 而weakmap 与 weakset 会 当对象 作为 null 是 集合里的key 也会对应的被清楚
    • 他们只有 add()
    • set()
    • get()
    • has()
    • delete() 方法 没有 keys() 遍历等
